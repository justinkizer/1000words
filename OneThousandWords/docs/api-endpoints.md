# API Endpoints

## HTML API

### Root

- `GET /` - loads React web app

## JSON API

### Users

- `POST /api/users`
- `PATCH /api/users`
- `GET /api/users`
- `GET /api/users/:id`

### Session

- `POST /api/session`
- `DELETE /api/session`

### Photos

- `GET /api/photos`
- `POST /api/photos`
- `GET /api/photos/:id`
- `DELETE /api/notes/:id`

### Followings

- `GET /api/followings`
- `POST /api/followings`
- `GET /api/followings/:id`
- `DELETE /api/followings/:id`
